Mapping Nietzsche

In Beyond Good and Evil, Friedrich Nietzsche includes a list of one hundred and twenty-three "Epigrams and Interludes." This graph shows those aphorisms mapped onto a two-dimensonal space. The coordinates are arbitrary, but the distance between each of the points indicates their similarity. The color of each aphorism (represented by its number) represents the cluster to which it belongs. Move your mouse over the number to display the full text of the aphorism.

Methodology

I began by parsing each aphorism and creating a word-frequency matrix, in which each row is an aphorism, each column a word, and each cell the number of times the word appears in the aphorism. Then, I calculate the tf-idf (term frequencyâ€“inverse document frequency), which measures the frequency of the word in each aphorism inversely weighted by how many times the word appears across all of the aphorisms. This results in a 123 x 633 matrix of aphorisms and words. I applied t-SNE (t-distributed stochastic neighbors embedding) for dimensonality reduction and used hiearchical clustering to create the colored groups.

Sources

I used the R packages tm for parsing and tsne for dimensonality reduction. The list of aphorisms can be found at Nietzsche's Features, which might be the only instance of a Nietzsche bibliography displayed in Comic Sans. Code and data are available on GitHub.
